# Architecture figures

PNG diagrams are **not** shipped in this bundle. Source graphs live in **`Aegis-contracts/architecture/`** as Graphviz `.dot` files.

To render PNGs locally (for docs or your own mirror):

```bash
cd Aegis-contracts
npm run architecture:render
```

The main dApp **How it works** page describes the client flow in text and links here instead of embedding images.
